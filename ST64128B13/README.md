# ST64128B13

## 参数

| 参数     | 属性                                                         |
| -------- | ------------------------------------------------------------ |
| 类型     | 单色STN点阵屏面板                                            |
| 分辨率   | 128x64+ICON                                                  |
| 尺寸     | 60mm x 42mm                                                  |
| 控制器   | UC1604                                                       |
| 接口     | 8-bit SPI                                                    |
| 有无背光 | 有                                                           |
| 工作电压 | 3.3V                                                         |
| 备注     | 支持电阻触控<br />需设置垂直镜像<br />需设置水平镜像<br />列偏移需设置为64 |
