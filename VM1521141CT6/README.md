# VM1521141CT6

## 参数

| 参数     | 属性                                                       |
| -------- | ---------------------------------------------------------- |
| 类型     | 单色STN点阵屏面板                                          |
| 分辨率   | 96x16                                                      |
| 尺寸     | 26mm x 8mm                                                 |
| 控制器   | ST7539                                                     |
| 接口     | I2C                                                        |
| 有无背光 | 有                                                         |
| 工作电压 | 3.3V                                                       |
| 备注     | SA0=1<br />SA1=1<br />扫描终止行需要设为15，否则对比度很低 |
