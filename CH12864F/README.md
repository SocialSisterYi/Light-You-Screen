# CH12864F

## 参数

| 参数     | 属性                                              |
| -------- | ------------------------------------------------- |
| 类型     | 单色STN点阵屏模组                                 |
| 分辨率   | 128x64                                            |
| 尺寸     | 54mm x 50mm（PCB）                                |
| 控制器   | ST7567                                            |
| 接口     | 6800 8-bit并口<br />8080 8-bit并口<br />8-bit SPI |
| 有无背光 | 有                                                |
| 工作电压 | 3.3V                                              |
| 备注     | -                                                 |
