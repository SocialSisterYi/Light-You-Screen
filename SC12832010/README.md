# SC12832010

## 参数

| 参数     | 属性                                |
| -------- | ----------------------------------- |
| 类型     | 单色STN点阵屏模组                   |
| 分辨率   | 128x32                              |
| 尺寸     | 70.5mm x 34mm（最宽处）             |
| 控制器   | ST7567                              |
| 接口     | 6800 8-bit并口<br />可改装8-bit SPI |
| 有无背光 | 有                                  |
| 工作电压 | 3.3V-5V                             |
| 备注     | -                                   |
