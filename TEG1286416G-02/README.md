# TEG1286416G-02

## 参数

| 参数     | 属性                 |
| -------- | -------------------- |
| 类型     | 单色STN点阵屏面板    |
| 分辨率   | 128x64               |
| 尺寸     | 59mm x 37.5mm        |
| 控制器   | UC1701               |
| 接口     | 8-bit SPI            |
| 有无背光 | 有                   |
| 工作电压 | 3.3V                 |
| 备注     | 扫描起始行需设置为32 |
