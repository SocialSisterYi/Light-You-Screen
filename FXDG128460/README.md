# FXDG128460

## 参数

| 参数     | 属性              |
| -------- | ----------------- |
| 类型     | 单色STN点阵屏面板 |
| 分辨率   | 128x64            |
| 尺寸     | 42mm x 36mm       |
| 控制器   | ST7567            |
| 接口     | 8-bit SPI         |
| 有无背光 | 无                |
| 工作电压 | 3.3V              |
| 备注     | 需设置垂直镜像    |
